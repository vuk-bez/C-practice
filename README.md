# C Practice – Zadaci

Ovaj repozitorijum sadrži riješene zadatke iz različitih oblasti programiranja u **programskom jeziku C**, uključujući:

- matrice
- nizove
- rekurziju
- stringove
- strukture podataka (**samo implemetacije**)
---
## Struktura repozitorija

- Svaki fajl sadrži implementaciju određenog zadatka.
- U svakom kodu je prije funkcije jasno naznačeno na koji zadatak se rješenje odnosi (komentarom).
---
## Materijali

Folder **`#Materijali`** sadrži:

- tekstove svih zadataka
- dodatne dokumente vezane za vježbe

---
## Napomena

Moguće je da se u nekim rješenjima nalaze greške. Repozitorijum služi prvenstveno kao materijal za učenje, pa se preporučuje dodatna provjera i testiranje koda.
